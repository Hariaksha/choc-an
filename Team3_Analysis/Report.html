Systems Operation Paragraph

Now that we have completed the requirements engineering phase of our software development lifecycle (see: Project 1 Report), we have determined what is necessary for the system to operate.

We will computerize the Provider Directory and Member/Provider account information, as well as processes to verify members, to bill ChocAn, and to run the main accounting procedure. The data will be stored on a Bitbucket cloud server. Our hardware includes terminals for providers, operators, and managers. Our data processing software interacts with the communications software, provider's bank accounts, the provider's terminal, Acme's Accounting software, and the EFT component. Managers interact with the system by requesting individual reports. Providers request provider directories, verify members, and bill ChocAn. Operators edit members/providers and email new privacy policies. Sending the privacy policy and editing members/providers satisfies our Security User Stories.
