Introduction

For our project, Chocoholics Anonymous, we served a customer company named ChocAn. With a subscription business model, ChocAn helps people break their addiction to chocolate. Our project group comprising of Ethan Busby, Adam Howard, Hariaksha Gunda, Ryan McCulley, Robert Read, Tyler Cruise, and James White spent 32.5 hours completing glossaries, use case diagrams, descriptions, matrices, user stories, and compiling files to create a succinct report on ChocAn's desired software product. The report was written in text documents and then compiled into a webpage using HTML and CSS programming languages.

Our timesheet kept track of work-hours contributed by each group member. This data was kept on a file named Hours.txt, then inputted into an HTML table upon the conclusion of the project. Similarly, our task distribution table was transferred from .txt to .html. This feature divvied our work into managable chunks and allowed each group member to know their exact responsibilities. Group members contributed to the project by 'pushing' their work to an online repository called Bitbucket -- this data can be found in the Bitbucket Statistics portion of our report. Adam Howard published a glossary that highlighted and defined key terms regarding ChocAn's software product. A use case diagram was conceived by Ethan Busby and modified by the group. This diagram visualizes the ChocAn software product functionality. Four actors are included in the schematic: a ChocAn member, provider, operator, and a nonhuman data center. Descriptions of each use case can be found in our attached file named ucdescriptions.pdf. Our story selection matrix was analyzed and completed by Tyler Cruise. Account lifestyle and accessible privacy policy user stories were written by Hariaksha Gunda, giving us a better look into the security details of the ChocAn software product. Finally, this introductory paragraph, as well as the webpage production, was completed by the project submitter, James White.
